# Just 10

Start small. Stop guilt-free.

Just 10 is a local-first mobile app for helping people begin avoided tasks with a gentle 10-minute timer.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router
- Zustand
- localForage
- date-fns
- lucide-react

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run build
npm run lint
```
